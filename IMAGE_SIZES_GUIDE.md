# 📸 Guide des Tailles d'Image Recommandées

Ce guide vous aide à télécharger des images optimisées pour votre boutique Flexiopage.

---

## 🎯 HERO IMAGE (Image de couverture principale)

### Desktop (Écran large)
- **Dimension**: 1920 x 600 px (ratio 16:10)
- **Poids maximal**: 500 KB
- **Format recommandé**: JPG ou WebP
- **Utilisation**: Page d'accueil, bannière principale
- **Conseil**: Image haute qualité avec texte visible

### Mobile (Téléphone)
- **Dimension**: 750 x 1000 px (ratio 3:4)
- **Poids maximal**: 300 KB
- **Format recommandé**: JPG ou WebP
- **Utilisation**: Affichage sur mobile (automatique)
- **Conseil**: Sujet centré, pas de texte sur les bords

### Exemple
```
Desktop:  1920 x 600 px = Full banner
Mobile:   750 x 1000 px = Vertical card
```

## 📦 IMAGES PRODUIT

### Image principale (carrée)
- **Dimension**: 1000 x 1000 px (ratio 1:1)
- **Poids maximal**: 400 KB
- **Format recommandé**: JPG ou PNG
- **Utilisation**: Grille produits, fiche produit
- **Conseil**: Fond blanc ou simple, produit centré

### Image miniature (thumbnail)
- **Dimension**: 500 x 500 px (ratio 1:1)
- **Poids maximal**: 150 KB
- **Format recommandé**: JPG ou WebP
- **Utilisation**: Aperçu rapide dans les listes
- **Conseil**: Même image que la principale, réduite

### Image zoom (haute résolution)
- **Dimension**: 2000 x 2000 px (ratio 1:1)
- **Poids maximal**: 800 KB
- **Format recommandé**: JPG ou WebP
- **Utilisation**: Zoom sur fiche produit
- **Conseil**: Détails bien visibles

### Galerie produit (carrousel)
- **Dimension**: 1200 x 1200 px (ratio 1:1)
- **Poids maximal**: 500 KB
- **Format recommandé**: JPG ou WebP
- **Quantité**: 3-6 images par produit
- **Conseil**: Différents angles, détails clairs

## ⚙️ OPTIMISATION CONSEILS

### Format d'image
- **JPG**: Photos, images complexes (compression avec perte)
- **PNG**: Logos, images avec transparence (sans compression)
- **WebP**: Meilleure compression (format moderne, compatible 95%)

### Compression
- Réduire la taille sans sacrifier la qualité
- Utiliser des outils: TinyPNG, ImageOptim, Squoosh
- Compression recommandée: 70-85% qualité JPG

### Dimensions
- Ne pas uploade une image plus grande que nécessaire
- Redimensionner AVANT upload (plus rapide, moins de bande passante)
- Utiliser des images carrées pour les produits (alignement)

### Texte dans l'image
- Éviter du texte petit (illisible sur mobile)
- Utiliser du texte via le CMS si possible
- Si texte dans image: font 24px minimum

## 🆘 PROBLÈMES COURANTS

### Image floue
- ✗ Uploadé une image trop petite compressée
- ✓ Utilise l'image native de haute qualité

### Image met longtemps à charger
- ✗ Image trop lourde (> limite recommandée)
- ✓ Compresse l'image avant upload

### Image coupée sur mobile
- ✗ Sujet sur les bords de l'image
- ✓ Laisse de la marge, sujet au centre

### Logo pixellisé
- ✗ Logo JPG au lieu de PNG
- ✓ Utilise PNG pour logo avec transparence
