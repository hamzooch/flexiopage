# Assets pour Flexiopage Desktop

## Icônes requises

Placez les fichiers PNG suivants dans ce dossier:

### icon.png (512x512)
Icône principale de l'application

### icon-tray.png (256x256)
Icône pour la barre des tâches/tray

## Icônes de notifications (optionnel)

Dossier `icons/`:
- `order.png` - Icône pour les commandes
- `message.png` - Icône pour les messages
- `payment.png` - Icône pour les paiements
- `alert.png` - Icône pour les alertes

Dimension recommandée: 128x128 ou plus

## Format

- **Format**: PNG avec fond transparent
- **Dimensions**: Voir au-dessus
- **Couleurs**: Compatible avec light/dark mode
